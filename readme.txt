Test Scores - VA SOL Branch

Version: 13.05

Author: Matt Walters

Overview:

A simple page to plug into the custom reports bundle.
Allows you to select a test and a score, and get the results for a selection of students.

VA SOL: Removes test score option and allows for multiselect of tests.

Installation: Upload the pages to /admin/reports/grading/

Requirements: You must have the custom reports bundle installed.
If you are not using dynamic report menu generation, you will need to add a link to the report manually.
